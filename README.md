# Bobby Media Photo Studio

Complete premium multi-page photography/event booking website with Next.js App Router, Tailwind CSS, Framer Motion, MongoDB, Mongoose, JWT Admin Auth, Admin Dashboard.

## Run
```bash
npm install
cp .env.example .env.local
npm run dev
```

Open: http://localhost:3000
Admin: http://localhost:3000/admin/login

## Instagram Images
Replace placeholder files/images in:
- public/images/hero/
- public/images/gallery/
- public/images/about/
- public/images/services/
- public/videos/hero-video.mp4

Use only Bobby Media Photo Studio Instagram assets.
